# Comprehensive Veterinary Laboratory Report Generator Application

Modify the existing veterinary biochemistry report generator application to include multiple types of veterinary laboratory reports. Create a multi-module system that can generate various types of veterinary diagnostic reports with standardized patient information and specialized examination fields.

## Application Structure

### 1. Main Dashboard & Report Type Selection
Create a main interface with report type selection:
- **Biochemistry Analyzer Report** (existing functionality)
- **Blood Smear Examination Report**
- **Wet Film Examination Report**
- **Skin Scraping Examination Report**
- **Impression Smear Examination Report**
- **Cytology Report**
- **Progesterone Test Report**

### 2. Standardized Patient Information Section (Common to All Reports)
All report types must include these mandatory fields:
- **Patient Name** (text input, required)
- **Parent Name** (text input for pet owner's name, required)
- **Species** (dropdown: Dog, Cat, Horse, Cattle, Sheep, Goat, Pig, Bird, Rabbit, Exotic, Other - required)
- **Breed** (text input, required)
- **Age** (number input with unit selector: years/months/weeks/days, required)
- **Weight** (number input with unit selector: kg/lbs/grams, required)
- **Date of Collection** (date picker, required)
- **Date of Report** (date picker, default to today, required)
- **Attending Veterinarian** (text input, required)

### 3. Report-Specific Modules

#### 3.1 Blood Smear Examination Report
**Test Parameters:**
- Red Blood Cell Morphology (dropdown: Normal, Anisocytosis, Poikilocytosis, Polychromasia, Other)
- White Blood Cell Count (number input)
- Platelet Count (number input)
- Parasites (dropdown: None detected, Babesia, Ehrlichia, Anaplasma, Other)
- Cell Distribution (text area)

**Examination Fields:**
- **Observation** (large text area for doctor's findings)
- **Advice** (text area for treatment recommendations)
- **Image Upload** (file upload with image optimization and cropping tool)

#### 3.2 Wet Film Examination Report
**Test Parameters:**
- Sample Type (dropdown: Urine, Vaginal discharge, Ear discharge, Other)
- Bacteria (dropdown: None, Cocci, Bacilli, Mixed, Other)
- Yeast/Fungi (dropdown: None detected, Candida, Malassezia, Other)
- Parasites (dropdown: None detected, Trichomonas, Other)
- Cells Present (checkboxes: Epithelial cells, WBCs, RBCs, Other)

**Examination Fields:**
- **Observation** (large text area)
- **Advice** (text area)
- **Image Upload** (file upload with optimization)

#### 3.3 Skin Scraping Examination Report
**Test Parameters:**
- Scraping Site (text input)
- Scraping Depth (dropdown: Superficial, Deep)
- Mites Detected (dropdown: None, Demodex, Sarcoptes, Cheyletiella, Other)
- Fungal Elements (dropdown: None, Dermatophytes, Spores, Hyphae)
- Secondary Infections (checkbox: Bacterial, Yeast)

**Examination Fields:**
- **Observation** (large text area)
- **Advice** (text area)
- **Image Upload** (file upload with optimization)

#### 3.4 Impression Smear Examination Report
**Test Parameters:**
- Sample Source (text input: wound, mass, discharge, etc.)
- Cell Types Observed (checkboxes: Neutrophils, Macrophages, Lymphocytes, Plasma cells, Other)
- Bacteria Present (dropdown: None, Intracellular, Extracellular, Mixed)
- Malignant Cells (dropdown: Not detected, Suspected, Present)
- Inflammatory Pattern (dropdown: Acute, Chronic, Granulomatous, Mixed)

**Examination Fields:**
- **Observation** (large text area)
- **Advice** (text area)
- **Image Upload** (file upload with optimization)

#### 3.5 Cytology Report
**Test Parameters:**
- Sample Type (dropdown: Fine needle aspirate, Fluid, Brush cytology, Other)
- Sample Site (text input)
- Cellularity (dropdown: Low, Moderate, High)
- Cell Types (checkboxes: Epithelial, Mesenchymal, Round cell, Other)
- Nuclear Characteristics (text area)
- Cytoplasm Characteristics (text area)
- Background Elements (text area)

**Examination Fields:**
- **Observation** (large text area)
- **Advice** (text area)
- **Image Upload** (file upload with optimization)

#### 3.6 Progesterone Test Report (Special Format)
**Test Parameters:**
- Progesterone Level (number input with unit ng/mL)
- Testing Method (dropdown: ELISA, Chemiluminescence, RIA, Other)
- Sample Quality (dropdown: Adequate, Suboptimal, Poor)

**Examination Fields (Specialized for Progesterone):**
- **Observation** (text area specifically for progesterone levels of examined dog)
- **Reference Range Table** (auto-populated based on breeding stage):
  ```
  Breeding Stage Reference Ranges:
  - Anestrus: < 1.0 ng/mL
  - Proestrus: 1.0 - 2.0 ng/mL
  - Estrus (LH surge): 2.0 - 5.0 ng/mL
  - Optimal breeding: 5.0 - 10.0 ng/mL
  - Post-ovulation: > 10.0 ng/mL
  - Pregnancy: 15.0 - 90.0 ng/mL
  ```
- **Advice** (radio button selection):
  - ○ Proceed (with breeding/insemination)
  - ○ Retest (specify timeframe)
- **Notes** (text area for additional doctor notes)
